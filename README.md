# Task Manager

## Description
Task management application developed with pure JavaScript, allowing users to create, validate, and organize tasks in real time.

The system includes duplicate prevention, data normalization, task movement between lists (pending and completed), and dynamic DOM manipulation.

## Features
- Create tasks
- Remove tasks
- Mark tasks as completed
- Move tasks between lists
- Prevent duplicate tasks

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open the `index.html` file in your browser
